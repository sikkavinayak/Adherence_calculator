Calculation Logic

1
Unique ID = To_Location + Excel_date_serial(From_Date) + Material — links each APO planned line to its actual YVR18 dispatch record
2
Check for inclusion (denominator) — APO rows where a truck was actually dispatched (actual qty > 0) AND planned Load Qty ≥ category threshold
3
Result = 1 (numerator) — included rows where actual dispatch ≥ 80% of planned Load Qty
4
Adherence % = Numerator / Denominator × 100, calculated per RDC
